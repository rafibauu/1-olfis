FAQ Website

Q : Bagaimana cara melakukan pendaftaran online ?

A : Pendaftaran online dapat dilakukan dengan mengunjungi link berikut ini.


Q : Bagaimana proses untuk mendapatkan username dan password akun website ini ?

A : Username dan password akan diberikan setelah anda melakukan pendaftaran dan pembayaran. Setelah itu juga anda akan mendapatkan nomor aktivasi untuk melakukan aktivasi akun.


Q : Bagaimana cara melakukan aktivasi akun ?

A : Masukkan nomer aktivasi yang diberikan oleh panitia setelah melakukan pembayaran. Nomer aktivasi akan dimintasi setelah log-in di website ini.


Q : Bagaimana jika saya tidak melakukan aktivasi akun ?

A : Beberapa fitur penting website tidak akan bisa digunakan jika aktivasi tidak dilakukan


Q : Bagaimana cara untuk mengubah data profil ?

A : Data profil dapat diubah dengan langsung mengubahnya pada halaman profil. Perhatikan kebenaran data, karena ada data yang tidak bisa diubah setelah pertama kali memasukkan data.
